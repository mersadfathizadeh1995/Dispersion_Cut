# DC Cut - Project Context

## Overview

**DC Cut** (Dispersion Cut) is a **Qt-based desktop application** for interactive editing, visualization, and analysis of **surface wave dispersion curves** from **MASW (Multichannel Analysis of Surface Waves)** and **passive seismic array** data.

### Domain
- **Geophysical Engineering / Near-Surface Geophysics**
- **Seismic wave analysis** for soil characterization
- Surface wave methods: Active MASW and Passive FK/FDBF analysis

### Primary Use Cases
1. Loading dispersion data from MATLAB `.mat`, CSV files, Geopsy `.max` (passive FK), or saved sessions
2. Interactive editing via box-select and delete for outlier picks
3. Filtering by velocity/frequency/wavelength ranges
4. Near-field evaluation using NACD (Normalized Array Coherence Distance) criteria
5. Averaging dispersion curves with statistical binning
6. Exporting cleaned curves to Geopsy-compatible TXT, CSV, or state files
7. Publication-quality figure generation with uncertainty envelopes

## Core Module (`dc_cut/core/`)

### `controller.py`
**Main Controller Class**

| Class | Description |
|-------|-------------|
| `InteractiveRemovalWithLayers` | Main application controller. Extends `BaseInteractiveRemoval`. Manages the matplotlib figure, axes (`ax_freq`, `ax_wave`), all data arrays, layer model, near-field evaluation, undo/redo history, and user action callbacks. Central orchestrator for all editing operations. |

**Key Methods:**
- `delete_selection()` - Remove points in selection boxes
- `perform_undo()` / `perform_redo()` - History navigation
- `apply_filters()` - Apply velocity/frequency/wavelength filters
- `compute_avg()` - Recalculate average curves
- `get_state()` / `set_state()` - Session persistence
- `start_nf_evaluation()` / `apply_nf_deletions()` - Near-field workflow

### `nearfield.py`
**Near-Field Evaluation**

| Class | Description |
|-------|-------------|
| `NearFieldInspector` | Computes NACD (Normalized Array Coherence Distance) per layer, flags points below threshold as near-field contaminated |

| Function | Description |
|----------|-------------|
| `compute_nacd()` | Calculate NACD = array_aperture / wavelength |
| `flag_nearfield_points()` | Mark points with NACD < threshold |

**NACD Formula:**
```
NACD = D / λ
where D = array aperture, λ = wavelength
Points with NACD < threshold (default 1.0) are near-field
```

## Data Conventions

### Array Storage
- All data stored as `list[np.ndarray]` (one array per offset/layer)
- `set_leg: list[str]` contains labels, with last two reserved for averages

### Physical Relationships
```
wavelength = velocity / frequency
λ = v / f

slowness (s/km) = 1000 / velocity (m/s)
s = 1000 / v
```

### Near-Field Criteria (NACD)
```
NACD = array_aperture / wavelength
NACD = D / λ

Points with NACD < threshold (default 1.0) are near-field contaminated
```

### Axis Scales
- **X-axis**: Logarithmic (frequency and wavelength)
- **Y-axis**: Linear (phase velocity)
- Auto-limits use percentile-based padding for outlier robustness
